How to Use

1. Import the CSS file:

import './RunningLights.css'

2. Add the class to your button:

<button className="running-light-border">Your Text</button>

3. For complex effects, add required HTML elements:

<button className="multi-directional">
  Your Text
  <div className="light-bottom"></div>
  <div className="light-left"></div>
</button>
